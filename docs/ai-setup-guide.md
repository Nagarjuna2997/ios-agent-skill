# AI Setup Guide — ios-agent-skill

> Just clone the repo. Every AI tool auto-detects it.

**Repo:** `https://github.com/Nagarjuna2997/ios-agent-skill.git`

## One-Line Install (Any Tool)

```bash
git clone https://github.com/Nagarjuna2997/ios-agent-skill.git
cd ios-agent-skill
```

That's it. Open your AI tool and start coding.

## Add to Any Existing Project (Universal)

Pick the file your AI tool reads and copy it:

```bash
# Clone once
git clone https://github.com/Nagarjuna2997/ios-agent-skill.git /tmp/ios-skill

# Then copy what you need:
cp /tmp/ios-skill/CLAUDE.md /your/project/           # Claude Code, Windsurf, Bolt
cp /tmp/ios-skill/AGENTS.md /your/project/           # Codex, Gemini, Amp, Lovable, Replit
cp /tmp/ios-skill/GEMINI.md /your/project/           # Gemini CLI, Antigravity
cp /tmp/ios-skill/.cursorrules /your/project/        # Cursor
cp /tmp/ios-skill/.clinerules /your/project/         # Cline
cp /tmp/ios-skill/.windsurfrules /your/project/      # Windsurf
cp /tmp/ios-skill/.rules /your/project/              # Zed AI

# Or copy directory-based rules:
cp -r /tmp/ios-skill/.cursor /your/project/          # Cursor (modern)
cp -r /tmp/ios-skill/.windsurf /your/project/        # Windsurf (modern)
cp -r /tmp/ios-skill/.roo /your/project/             # Roo Code
cp -r /tmp/ios-skill/.amazonq /your/project/         # Amazon Q
cp -r /tmp/ios-skill/.trae /your/project/            # Trae
cp -r /tmp/ios-skill/.aiassistant /your/project/     # JetBrains AI
cp -r /tmp/ios-skill/.junie /your/project/           # Junie
cp -r /tmp/ios-skill/.continue /your/project/        # Continue.dev / PearAI
cp -r /tmp/ios-skill/.augment /your/project/         # Augment Code
cp -r /tmp/ios-skill/.tabnine /your/project/         # Tabnine
cp -r /tmp/ios-skill/.kilocode /your/project/        # KiloCode
mkdir -p /your/project/.github && cp /tmp/ios-skill/.github/copilot-instructions.md /your/project/.github/  # Copilot
```

```

```

If it uses `Color(hex: "#0077B6")` and references the design system — the skill is loaded.
